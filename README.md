# Catoosa First Responder Activity

This repository hosts the public site for **Catoosa First Responder Activity**.

## Files
- `index.html` — Homepage (scanner, auto news, tip form).
- `about.html` — About page with banner and mission.

## Edit Instructions
- On GitHub, open a file and click the pencil icon to edit, then **Commit changes**.
- To change the news sources, edit the `FEEDS` array near the bottom of `index.html`.
